 Retail Sales Analysis & Business Insights

 Objective
Analyze retail sales data to identify key factors influencing sales performance.

 Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

 Key Insights
- Top 5 item types contribute highest sales
- Supermarket Type1 performs best
- Tier 3 locations generate strong revenue
- Low-fat products dominate sales

 Business Recommendations
- Focus on high-performing product categories
- Expand outlets in Tier 3 locations
- Optimize inventory based on demand
